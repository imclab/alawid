7DRL

(This page is WIP)

Made for the 7 Day Roguelike Challenge 2011.

The game is played inside a web browser and requires WebGL support, so you need a rather new browser (e.g. Google Chrome 9+ or Firefox 4 - IE won't work). Also, your graphics card needs to be at least OpenGL 2 capable (with decent drivers) for WebGL to work. You can check supported browsers and other notes in Khrono's WebGL Wiki.

The game is located in the subdirectory called game.

Note that running the game locally will not probably work. You need a webserver to answer the XMLHttpRequests that are used for loading shaders from dedicated files (or at least that is the explanation I thought the most likely).
